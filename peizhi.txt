services选择
Service 类型	用途	适合你吗？	说明
🟢 Web Service	部署持续运行的 Web 服务器（如 Node.js、Python、Go 后端）	
✅ 选这个！	你的 Express 后端就属于这种，需要一直在线，接收请求

🟡 Static Site	部署纯静态网站（HTML/CSS/JS，无后端逻辑）		
比如你只想放一个  index.html  页面，不需要 API，才用它
🔵 Private Service	部署只能在内部网络访问的服务（不暴露公网）		
比如内部工具、数据库代理，外部无法直接访问
🟠 Background Worker	部署长时间运行的后台任务（无 HTTP 请求）		
比如定时清理数据、发邮件队列，不需要对外提供接口
🟣 Cron Job	部署定时执行的任务（比如每天凌晨跑一次）		
比如每天备份数据库、生成报表
🔴 Redis / PostgreSQL	部署数据库或缓存服务	（现阶段）	
你以后可能需要，但现在先部署后端
